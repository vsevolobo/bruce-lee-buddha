# Installation Guide for Bruce Lee Buddha

This guide provides step-by-step instructions for installing the Bruce Lee Buddha project on various platforms.

## 1. Progressive Web App (PWA) on Web
- Open your preferred web browser.
- Navigate to the link: `https://your-app-url.com`
- Click on the install icon (usually shown in the address bar).
- Follow the prompts to complete the installation.

## 2. PWA on Android
- Open the Chrome browser on your Android device.
- Navigate to the link: `https://your-app-url.com`
- Tap the menu icon (three dots in the top right corner).
- Select "Add to Home screen."
- Follow the prompts to install the PWA.

## 3. Node.js Server Setup
- Ensure you have [Node.js](https://nodejs.org/) installed on your machine.
- Clone the repository:
  ```bash
  git clone https://github.com/vsevolobo/bruce-lee-buddha.git
  cd bruce-lee-buddha
  ```
- Install dependencies:
  ```bash
  npm install
  ```
- Start the server:
  ```bash
  npm start
  ```
- The server will start on `http://localhost:3000`.

## 4. Android App Installation from APK
- Download the APK file from the releases section of the GitHub repository.
- On your Android device, navigate to Settings > Security.
- Enable "Unknown Sources" to allow installation from external sources.
- Open the downloaded APK file and tap "Install."
- Follow the instructions to complete the installation.

## 5. Installing from Google Play (if applicable)
- Open the Google Play Store on your Android device.
- Search for "Bruce Lee Buddha."
- Tap on the app and click "Install."
- Wait for the installation to complete. Open the app from your app drawer.

## 6. F-Droid Installation
- Open the F-Droid app on your Android device.
- Search for "Bruce Lee Buddha."
- Tap on the app and click "Install."
- Follow the on-screen instructions to complete the installation.

Feel free to reach out with any questions or issues while installing!
